# Работа с GitHub репозиторием

## ✅ Репозиторий подключен

Локальный проект подключен к репозиторию: https://github.com/freedomdp/bambu_service_bot

## Как делать коммиты и отправлять изменения

После внесения изменений в код выполните следующие команды:

### 1. Проверьте статус изменений:
```bash
git status
```

### 2. Добавьте измененные файлы:
```bash
# Добавить все изменения
git add .

# Или добавить конкретные файлы
git add src/main.py src/handlers/commands.py
```

### 3. Создайте коммит с описанием изменений:
```bash
git commit -m "описание того, что было доработано"
```

**Примеры хороших сообщений коммитов:**
- `feat: добавлена валидация email при создании заявки`
- `fix: исправлена ошибка при загрузке фото`
- `refactor: улучшена структура обработчиков команд`
- `docs: обновлена документация в README`

### 4. Отправьте изменения на GitHub:
```bash
git push
```

## Рекомендации по коммитам

Используйте понятные сообщения, которые описывают:
- **Что** было изменено
- **Зачем** это было сделано (если не очевидно)

Формат: `тип: описание`

**Типы коммитов:**
- `feat` - новая функциональность
- `fix` - исправление ошибки
- `refactor` - рефакторинг кода
- `docs` - изменения в документации
- `style` - форматирование кода
- `test` - добавление тестов
- `chore` - обновление зависимостей, конфигурации

## Просмотр истории коммитов

```bash
# Последние 10 коммитов
git log --oneline -10

# Подробная история
git log
```

## Если нужно отменить изменения

```bash
# Отменить изменения в файле (до добавления в staging)
git checkout -- имя_файла.py

# Отменить последний коммит (сохранив изменения)
git reset --soft HEAD~1
```
