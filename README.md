# Bambu Service Bot

Telegram бот для сервісного центру Bambu Lab Україна.

## Функціонал
- Створення заявок на сервісне обслуговування
- Збір інформації про принтер та проблему
- Завантаження фото та скріншотів
- Відправка заявок інженеру
